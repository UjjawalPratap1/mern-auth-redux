# mern-auth-redux
A full-stack MERN authentication system with Login &amp; Registration using React.js, Node.js, Express.js, MongoDB, and Redux Toolkit. Features protected routes, persistent login state, and responsive UI with Tailwind CSS.

# MERN Auth Redux

A full-stack authentication system built with the MERN stack.

## Features
- User Registration & Login
- Protected Routes
- Redux Toolkit for global state
- Persistent login using localStorage
- React Router navigation
- Tailwind CSS modern UI
- MongoDB for user data
- Express & Node backend APIs

## Tech Stack
- React.js
- Redux Toolkit
- Node.js
- Express.js
- MongoDB
- Tailwind CSS

## How to Run

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start
